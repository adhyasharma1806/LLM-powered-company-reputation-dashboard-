AI Chart Explainer 📊🤖
📌 Project Overview
AI Chart Explainer is a simple data visualization project that automatically explains charts in natural language using a Large Language Model (LLM).
The goal is to help users understand trends, patterns, and insights from charts without manually analyzing raw data.

This project is designed as part of the Emerging Tools & Technology (CSE3232) lab work.

🎯 Objectives
To visualize data using standard charts
To automatically generate human-readable explanations for charts
To demonstrate the basic integration of data visualization with LLMs
To improve interpretability of visual data
🛠️ Technologies Used
Python
Pandas – data handling
Matplotlib / Seaborn – chart generation
Streamlit – user interface
LLM API / Rule-based Text Generator – chart explanation
⚙️ System Workflow
Load dataset (CSV or predefined data)
Generate chart (bar / line / pie)
Extract basic statistics from data
Send chart summary to LLM
Display chart along with AI-generated explanation
📊 Features
Supports common chart types (Bar, Line, Pie)
Automatically explains:
Trends
Highest and lowest values
Overall observations
Beginner-friendly and easy to extend
Suitable for lab demonstrations and viva
